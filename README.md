## How to use

[Download .vimrc](https://raw.githubusercontent.com/ashfinal/vimrc-config/master/.vimrc) and put it in your home directory. Run vim as usual.

- *nix & Mac

    `curl -Lo ~/.vimrc http://jump.im/re`

- Windows

    You will figure it out. :D
